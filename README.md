## 安装

1. 下载 [extension.zip](./extension.zip)，解压到一个固定位置（如 `D:\Tools\FliggyToolbox`）
2. Chrome 地址栏输入 `chrome://extensions/` 并回车
3. 右上角开启**开发者模式**
4. 点击**加载已解压的扩展程序** → 选择解压出来的文件夹 → 确定
5. 刷新工作页面，安装完成

> 解压的文件夹不要移动或删除，否则扩展会失效。

---

## 更新

当有新版本时，扩展会弹出通知：**"飞猪客服工具箱有新版本"**

1. 点击通知，自动打开本仓库页面
2. 在页面中点击 [extension.zip](./extension.zip) 下载最新版本
3. 解压，覆盖之前的文件夹
4. 打开 `chrome://extensions/`，点击扩展卡片上的刷新按钮即可生效
